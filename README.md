Design System
Colors
Primary: Black/Dark (#000000) - Sophisticated luxury feel

Success: Emerald (#059669) - Completed/delivered states

Warning: Amber (#D97706) - Low stock/pending states

Danger: Red (#DC2626) - Cancelled/out of stock

Info: Blue (#2563EB) - Ordered/processing states

Accent: Violet (#7C3AED) - Featured products

Typography
Font: Inter (Google Fonts)

Weights: 400, 500, 600, 700, 800, 900

Scale: 9px - 32px for hierarchy

Components
Cards: rounded-2xl / rounded-3xl with borders and shadows

Buttons: Pill-shaped (rounded-full) with hover transitions

Inputs: Rounded with focus states

Badges: Color-coded pills for status indicators

🔐 Security
Row Level Security (RLS): Database-level access control

Admin Protection: Role-based access to admin dashboard

Supabase Auth: Secure authentication with email/password

Environment Variables: Sensitive keys never exposed in code

Input Validation: Server-side validation on all mutations

📱 Responsive Design
The admin dashboard is fully responsive:

Desktop (1024px+): Full sidebar + detailed tables

Tablet (768px - 1023px): Collapsible sidebar, 2-column grids

Mobile (< 768px): Single column, dropdown navigation, touch-optimized
